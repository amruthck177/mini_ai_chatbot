# 🦸 Contributing to ZAPTRON

First off, thanks for taking the time to contribute! ZAPTRON is only as strong as his league of developers. ⚡

## 🛡️ Code of Conduct

As a hero in this project, we expect everyone to follow our standard of excellence:
- Be respectful and encouraging.
- Focus on positive, constructive feedback.
- Help others grow their technical power-sets!

## 🚀 How Can I Help?

### 1. Reporting Bugs (Finding the Villains)
If you spot a bug (a glitch in the Digital Universe), please open an Issue. Include:
- A clear title.
- Steps to reproduce the villainous behavior.
- What you expected vs. what actually happened.

### 2. Suggesting Features (New Superpowers)
Have an idea for a new feature? 
- Open an Issue with the [Feature] tag.
- Explain why this superpower would be useful for ZAPTRON.

### 3. Pull Requests (Joining the Mission)
1. Fork the repository.
2. Create your Feature Branch (`git checkout -b feature/AmazingNewPower`).
3. Commit your changes (`git commit -m 'Add some AmazingNewPower'`).
4. Push to the branch (`git push origin feature/AmazingNewPower`).
5. Open a Pull Request!

## 🎨 Style Guidelines

- **Keep it Heroic**: Maintain the comic book aesthetic in all UI changes.
- **Stay Modular**: Keep the React components clean and focused.
- **Gemini First**: Ensure all AI logic remains compatible with the Google Gemini API.
