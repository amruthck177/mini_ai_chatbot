# ⚡ ZAPTRON: Guardian of the Digital Universe! 🦸

Welcome to the **ZAPTRON Comic AI Chatbot**, an immersive, high-fidelity AI companion built for the Google Gemini API. ZAPTRON isn't just a chatbot; he's a heroic entity designed to assist you with wit, flair, and classic comic book energy!

![Zaptron Interface](https://via.placeholder.com/800x450.png?text=ZAPTRON+COMIC+INTERFACE)

## 🌟 Features

- **💥 Comic Book Aesthetic**: Full pop-art design system with halftone patterns, ink-heavy borders, and dynamic SFX bursts.
- **🤖 Powered by Gemini**: Integrates with Google Gemini 1.5 Flash, 1.5 Pro, and 2.0 Flash models.
- **⚡ Interactive Mood System**: ZAPTRON's avatar and expressions change based on the conversation's context.
- **🛠️ Multi-Model Selector**: Switch between models in real-time to manage your API quotas.
- **🌓 Dark Mode**: Toggle between Day-Shift Heroics and Night-Shift Vigilance.
- **📋 Heroic Utilities**: Quick-action power buttons, message copying, and reaction tracking.

## 🚀 Mission Setup (Local)

To deploy ZAPTRON in your local sector:

1. **Clone the Fortress**:
   ```bash
   git clone https://github.com/amruthck177/mini_ai_chatbot.git
   cd mini_ai_chatbot
   ```

2. **Supply Dependencies**:
   ```bash
   npm install
   ```

3. **Fuel the Reactor**:
   Create a `.env` file in the root and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```
   *Get a free key at [Google AI Studio](https://aistudio.google.com).*

4. **Ignition**:
   ```bash
   npm run dev
   ```
   Open **http://localhost:5173** to meet ZAPTRON!
